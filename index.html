O que é JavaScript?

JavaScript é uma linguagem de programação que adiciona interatividade ao seu site. Ele permite ações como resposta a cliques de botões, validação de formulários, mudanças dinâmicas na página, animações e muito mais. Junto com HTML (que estrutura o conteúdo) e CSS (que define o estilo), o JavaScript dá vida ao site, tornando-o dinâmico e interativo. Apesar do nome parecido, JavaScript (JS) e Java são linguagens completamente diferentes. Enquanto o Java é amplamente utilizado no desenvolvimento de aplicativos, softwares corporativos e dispositivos móveis, o JavaScript é voltado principalmente para a programação web.

Trabalhando com JavaScript

Para manipular JavaScript, é ideal que você já tenha conhecimento de HTML e CSS. Vamos começar criando um alerta simples, que pode ser incorporado diretamente no HTML usando a tag <script>. Por exemplo:

HTML 

<script>alert("Olá, Mundo!");</script>

No entanto, um alerta constante tornaria o site pouco funcional. Além disso, escrever scripts diretamente no HTML pode deixar o código desorganizado e difícil de manter. Por isso, o mais recomendado é criar um arquivo externo para o JavaScript, como na maioria dos casos. Para carregar esse arquivo no HTML, utilizamos a seguinte sintaxe:

HTML 

<script>js/script.js</script>

Aqui: js/ é a pasta onde o arquivo JavaScript está armazenado (o nome da pasta pode ser qualquer um). script.js é o arquivo JavaScript, que também pode ter qualquer nome, desde que tenha a extensão .js.

Agora vamos chamar um texto escrito no arquivo JS, existem duas formas:

HTML 

<p><script>
document.write("<i>A primeira é escrevendo dentro do HTML com JavaScript document.write()</i>");
</script></p>

O método document.write() escreve conteúdo diretamente na página HTML enquanto ela está sendo carregada.

E a outra forma é como devemos trabalhar, chamando o texto que está dentro do arquivo .js

HTML 

<p id="texto"></p>

Essa ainda é a melhor forma de se trabalhar, não correndo o risco de sobrescrever outros códigos.

JS 

document.getElementById("texto").innerHTML = "<i>Meu primeiro texto com <b>JS</b>!</i>";

Veja o resultado logo baixo:

Explicação:

document → Representa a página HTML carregada no navegador. Ele permite acessar e manipular os elementos do documento.

.getElementById("texto") → Busca um elemento HTML que tenha o atributo id="texto".

Se no HTML houver <p id="texto"></p>, esse método encontra esse parágrafo.

Esse comando substitui tudo dentro da tag selecionada.

= "Meu primeiro texto com JS!"; → Define um novo conteúdo para o elemento HTML.

Se antes a tag <p id="texto"></p> estava vazia, ela agora terá o texto "Meu primeiro texto com JS!".

Para finalizarmos essa parte, vamos ver o console.log(). Tem a mesma função do alert(); porém é os resultados são apresentados apenas no console do navegador.

O que é uma variável?

Uma variável é um espaço na memória do computador usado para armazenar valores, que podem ser de diversos tipos, como números, textos, objetos, funções e muito mais.

No JavaScript, as variáveis são declaradas utilizando as palavras-chave var, let ou const, cada uma com suas particularidades:

var – Pode ser redeclarada e não é limitada ao escopo de bloco.

let – Não pode ser redeclarada e é restrita ao escopo de bloco.

const – Não pode ser redeclarada, é restrita ao escopo de bloco e seu valor não pode ser alterado após a atribuição.

Entender essas diferenças é essencial para escrever códigos mais organizados e eficientes!

Declaração de variáveis

A declaração pode receber o nome que quiser desde que não contenha carácters especiais como "@", não comece com letras maiúsculas não seja separados por traço como "nome-sobrenome" e nem contenha espaço entre os nomes, no cajo o ideal seria "nomeSobrenome".

Podemos declarar as variáveis da seguinte forma:

var nome

var sobrenome

var nomeCompleto

ou conforme abaixo que é mais fluido e prático.

JS

Operadores de Incremento e Decremento

Os operadores de incremento ++ adicionam 1 número a mais ao valor numérico da variável, já o de decremento -- subtrai 1 número do valor da variável.
Isso não apenas soma ou subitrai um número, mas também altera o valor da variável, sento assim a variável que antes valia "x", agora passa a valer "y".

    `<p style="padding-right: 2rem;"><b>valorIncremento</b> =${valorIncremento = ++valor3}</p>`;

Operadores de Comparação

Os operadores de comparação são usados para comparar valores e retornam um valor booleano (true ou false). Eles são essenciais para a tomada de decisões no código, como em estruturas condicionais (if, while).
Vamos ver um exemplo de código e respectivamente o resultado aqui no html:

JS 

    var valorComp1 = 10, valorComp2 = 5, valorComp3 = "5";

document.getElementById("operadores_comparacao").innerHTML =
    `<p style="padding-right: 2rem; text-align: center;">10 <b>==</b> 5<br>
    ${valorComp1 == valorComp2}</p>`;

document.getElementById("operadores_comparacao").innerHTML +=
    `<p style="padding-right: 2rem; text-align: center;">5 <b>===</b> "5"<br>
    ${valorComp1 === valorComp3}</p>`;

document.getElementById("operadores_comparacao").innerHTML +=    
    `<p style="padding-right: 2rem; text-align: center;">10 <b>!=</b> 5<br>
    ${valorComp1 != valorComp2}</p>`;

document.getElementById("operadores_comparacao").innerHTML +=    
    `<p style="padding-right: 2rem; text-align: center;">10 <b>!==</b> 5<br>
    ${valorComp1 !== valorComp2}</p>`;

document.getElementById("operadores_comparacao").innerHTML +=
    `<p style="padding-right: 2rem; text-align: center;">10 <b>></b> 5<br>
    ${valorComp1 > valorComp2}</p>`;

document.getElementById("operadores_comparacao").innerHTML +=
    `<p style="padding-right: 2rem; text-align: center;">10 <b><</b> 5<br>
    ${valorComp1 < valorComp2}</p>`;

document.getElementById("operadores_comparacao").innerHTML +=
    `<p style="padding-right: 2rem; text-align: center;">10 <b>>=</b> 5<br>

    `<p style="padding-right: 2rem; text-align: center;">10 <b><=</b> 5<br>
    ${valorComp1 <= valorComp2}</p>`;

Note que há apenas uma comparação onde 5 é comparado com ele mesmo, isso foi feito para compara se além de serem iguais no valores, se eles são equivalentes, no caso um dos valores é um texto, pois a variável com "5" está com aspas e tornando-a um texto.

Vamos entender as funções

Uma função no JavaScript é um bloco de código projetado para executar uma tarefa específica.
É como uma pequena "fábrica" onde você faz uma entrada e ele dá uma saída.
Pode ser encarado como um "mini-programa" projetado para fazer uma tarefa que vai contribuir com todo código.
Uma função é executada como algo a invoca, dando uma ordem de ação.

Função de Somar

Criamos uma função para somar duas variáveis determinadas com os valores 5 e 2.

JS 

var valor5 = 5, valor6 = 2;

}

document.getElementById("funcao_somar").innerHTML =
    `A soma de 5+2 é = ${somar(5, 2)}`;

Nesse momento a função não precisou de uma ação externa para ser executada, mas ela já conta com uma ordem para executar a mesma.

Vamos entender os Eventos

Eventos são interações ou ocorrências que acontecem em uma página HTML — como um clique em um botão, o movimento do mouse sobre um elemento, o envio de um formulário, entre outros.
No JavaScript, é possível "ouvir" esses eventos e reagir a eles executando funções específicas. Esse processo é chamado de manipulação de eventos.
Ou seja, quando um evento acontece, o JavaScript pode executar uma ação automaticamente, como exibir uma mensagem, alterar um conteúdo ou aplicar uma animação.

Os eventos mais comuns são:

→ onclick - Ocorre quando um elemento é clicado.

→ ondblclick - Ocorre quando há um duplo click.

→ onmouseover - Ocorre quando o mouse passa sobre um elemento.

→ onmouseout - Ocorre quandmouseOvero o mouse sai de um elemento.

→ onmousemove - Ocorre quando o mouse se move sobre um elemento.

→ onmousedown - Ocorre quando um botão do mouse é pressionado.

→ onmouseup - Ocorre quando um botão do mouse é liberado.

→ onfocus - Ocorre quando um elemento recebe o foco.

→ onblur - Ocorre quando um elemento perde o foco.

→ onchange - Ocorre quando o valor de um elemento é alterado.

→ onkeydown - Ocorre quando uma tecla é pressionada.

→ onkeypress - Ocorre quando uma tecla é pressionada e liberada.

→ onkeyup - Ocorre quando uma tecla é liberada.

→ onload - Ocorre quando um documento é carregado.

→ onresize - Ocorre quando a janela do navegador é redimensionada.

Evento de clique

O evento de clique acontece quando o usuário clica em um elemento da página, como um botão, por exemplo. É um dos eventos mais comuns em interações com páginas HTML, muito utilizado para enviar formulários, acionar funções ou exibir mensagens.

Vamos criar um exemplo simples para demonstrar seu funcionamento. Em muitos casos, utilizamos o console.log() para visualizar o resultado ou testar o comportamento do evento de forma prática durante o desenvolvimento.

function clique() {
    alert("Você clicou no botão!");
}

Com o código criado, agora precisamos criar o botão para que o evento de clique aconteça.

Eventos onmouseover e onmouseout

Os eventos onmouseover e onmouseout são utilizados para detectar quando o mouse entra ou sai de um elemento da página.

Esses eventos são muito usados para criar efeitos visuais, como mudar a cor de um botão, exibir uma dica ou destacar uma área interativa da interface..

Vamos criar um código para testar o efeito desejado com as duas ações.

function mouseOver() {
    let div = document.getElementById("OnMouse");

}

Eventos onmousemove, onmousedown e onmouseup

OOs eventos onmousemove, onmousedown e onmouseup são usados para detectar interações mais detalhadas do mouse com os elementos de uma página. Com onmousemove, é possível capturar o movimento do mouse enquanto ele se desloca sobre um elemento, permitindo, por exemplo, criar efeitos de arrastar ou até mostrar as coordenadas em tempo real. Já o onmousedown é disparado no momento em que o botão do mouse é pressionado sobre um elemento, enquanto o onmouseup é acionado quando o botão do mouse é solto. Esses eventos funcionam muito bem em conjunto, especialmente para criar ações como clicar e arrastar ou desenhar elementos na tela.

JS 

function mouseMove() {
    let div = document.getElementById("OnMouse1");
    div.style.backgroundColor = "blue";
    let txt = document.getElementById("OnMouse1");

    let ptxt = document.getElementById("OnMouse1").innerHTML = "Moveu o mouse!
Pressione";
}

function mouseDown() {
    let div = document.getElementById("OnMouse1");
    div.style.backgroundColor = "green";
    let txt = document.getElementById("OnMouse1");
    txt.style.color = "white";
    let ptxt = document.getElementById("OnMouse1").innerHTML = "Você pressionou o mouse!";
}

function mouseUp() {
    let div = document.getElementById("OnMouse1");
    div.style.backgroundColor = "";
    let txt = document.getElementById("OnMouse1");
    txt.style.color = "";
    let ptxt = document.getElementById("OnMouse1").innerHTML = "Você soltou o mouse!";
}

Eventos onfocus e onblur

Os eventos onfocus e onblur são muito úteis para lidar com campos de formulário. O evento onfocus é ativado quando um elemento, como um campo de texto, ganha o foco — ou seja, quando o usuário clica ou navega até ele, permitindo a digitação. Já o onblur ocorre quando esse mesmo campo perde o foco, seja porque o usuário clicou fora dele ou navegou para outro elemento. Esses eventos são comumente utilizados para destacar campos ativos, validar dados em tempo real ou esconder e mostrar dicas de preenchimento conforme o usuário interage com o formulário.

JS 

function campoFocado() {
    document.getElementById("onFocus").value = "";
    let div = document.getElementById("onFocus");
    div.style.backgroundColor = "yellow";
    div.style.color = "black";
}

function campoDesfocado() {
    document.getElementById("onFocus").value = "Clique para apagar";
    let div = document.getElementById("onFocus");
    div.style.backgroundColor = "";
    div.style.color = "";

}

Eventos onkeydown, onkeypress e onkeyup

Os eventos onkeydown, onkeypress e onkeyup estão relacionados à interação do teclado com os elementos da página. O onkeydown é disparado assim que uma tecla é pressionada, ou seja, no momento em que o usuário começa a pressionar uma tecla. Já o onkeypress também é acionado durante a pressão de uma tecla, mas geralmente está associado a teclas que geram caracteres (embora esse evento esteja sendo cada vez menos utilizado, pois não captura todas as teclas). Por fim, o onkeyup é ativado quando a tecla é solta, ou seja, após o pressionamento. Esses eventos são úteis para detectar digitação em tempo real, criar atalhos de teclado ou validar campos conforme o usuário digita.

JS 

function teclaPressionada() {
    let tecla = document.getElementById("onFocus1");
    tecla.style.backgroundColor = "#c40000";
}

function teclaPressionadaELiberada() {
    let tecla = document.getElementById("onFocus1");
    tecla.style.backgroundColor = "#20b200";
    tecla.style.color = "white";
}

function teclaLiberada() {
    let tecla = document.getElementById("onFocus1");
    tecla.style.backgroundColor = "#000d89";
}

Como funciona os Arrays

Arrays são objetos que armazenam várias informações em uma única variável. No JavaScript, os arrays são declarados com colchetes [ ] e podem conter qualquer tipo de valor, como números, textos, objetos, funções, etc.
Os arrays são indexados, ou seja, cada elemento do array possui um índice que começa em 0.
Os arrays podem ser acessados e manipulados através dos seus índices.

Criamos um Array logo abaixo com frutas, veja que foi manipulado e há algumas explicações dentro do próprio código.

JS 

 function campoAlterado() {
    alert("Os valores foram alterados!");
} //Esse evento é chamado quando o campo é alterado.

const frutas = ["Banana", "Maçã", "Morango", "Uva", "Pera"];
frutas[0] = "Abacaxi"; //Alterando o valor do índice 0, que era Banana agora é Abacaxi
frutas[5] = "Melancia"; //Adicionando um novo valor no índice 5
frutas.push("Laranja"); //Adicionando um novo valor no final do array

document.getElementById("array").innerHTML =
`Na salada de fruta contém: ${frutas}`;

O que são Declarações Condicionais

As declarações condicionais são usadas para tomar decisões com base em diferentes condições. No JavaScript, as declarações condicionais são feitas com a palavra-chave if, else if e else.
Se a condição for verdadeira, o bloco de código dentro do "if" será executado.
Se a condição for falsa, o bloco de código dentro do "else" será executado.

Criamos uma variável hora do sistema, que é exatamente a hora do seu computador, e a variável saudadacao para retornar qual tipo de saudação é o mais adequado. dentro do próprio código.

JS 

var hora = new Date().getHours();
var saudacao;

if (hora < 12) {
    saudacao = "Bom dia!";
}
else if (hora < 18) {
    saudacao = "Boa tarde!";
}
else {
    saudacao = "Boa noite!";
}

document.getElementById("horaSistema").innerHTML = (saudacao);

Declarações Condicionais com funções

Um exemplo mais prático para entendermos a aplicação da declaração condicional, é a validação de preenchimento de um campo, para isso vamos criar um campo onde se deve digitar o nome corretaente.

JS 

function veriNome() {
    let nomeCondicional = document.getElementById("nomeCond").value;

    if  (nomeCondicional == "" || nomeCondicional == null) {
        let p = document.getElementById("resultNome")
        p.innerHTML = "Digite um nome!";
        p.style.color = "red";
    }

    else if (nomeCondicional.length < 3) {
        let p = document.getElementById("resultNome")
        p.innerHTML = "O nome deve ter mais de 3 caracteres!";
        p.style.color = "orange";
    }

    else {
        let p = document.getElementById("resultNome")
        p.innerHTML = `Ok, está completo`;
        p.style.color = "green";
    }
}

if & else são condicionais mais utilizados no JavaScript para verificação de campos, como pode ver o resultado abaixo, se o campo está vazio, então o resultado é falso. Se o campo está preenchido corretamente, então o resultado é verdadeiro.

Como funciona um Switch

A declaração switch é usada para selecionar um dos muitos blocos de código a serem executados. A declaração switch avalia uma expressão, combina o valor da expressão com um caso e executa o bloco de código associado ao caso.

Criamos uma variável para retornar o dia da semana, essa informação é exatamente a do seu computador.

JS 

function diaSemana() {
    let dia;
    switch (new Date().getDay()) {
        case 0:
            dia = "Hoje é Domingo";
            break;
        case 1:
            dia = "Hoje é Segunda-feira";
            break;
        case 2:
            dia = "Hoje é Terça-feira";
            break;
        case 3:
            dia = "Hoje é Quarta-feira";
            break;
        case 4:
            dia = "Hoje é Quinta-feira";
            break;
        case 5:
            dia = "Hoje é Sexta-feira";
            break;
        case 6:
            dia = "Hoje é Sábado";
            break;
        default:
            dia = "Não sei qual é o dia";

    }
    document.getElementById("switchCase").innerHTML = (dia);
}

Vamos entender os Laços de Repetição

Os laços de repetição são usados para executar o mesmo bloco de código várias vezes.
No JavaScript, os laços de repetição são feitos com as palavras-chave for, while e do while.

Vamos criar uma variável chamada i de incersão, e nela vamos dizer enquanto a variável for menor que 10, o resultado será exibido.

JS 

for (let i = 0; i < 10; i++) {
    document.getElementById("repeticao").innerHTML += i + ", ";
}

Como pode ver o resultado é encerrando antes de chegar ao 10, pois dizemos que enquanto menor, no próximo exemplo vamos fazer diferente, e o resultado será diferente também.

Vamos entender os Eventos de Tempo

Os eventos de tempo são usados para executar funções após um determinado tempo.
No JavaScript, os eventos de tempo são feitos com as funções setTimeout e setInterval.

As funções de tempo são especificadas em milisegundos: 1000 ms = 1 segundo.

Vamos criar um exemplo onde o tempo será contando de 0 a 5 ao clicar no botão "Ativar contagem", a mensagem "Começou a contar!" aparece na tela e ao chegar ao seu final, a mensagem "Executou a contagem" é exibida na tela.
Caso seja clicado no botão "Parar contagem" seja clicado, a mensagem "Parou a contagem!" será exibida e zera a contagem.

JS 

function starCount() {
        document.getElementById("time").innerHTML = "Começou a contar!";
        tempo = setTimeout(function() { 
            document.getElementById("time").innerHTML = "Executou a contagem";
         }, 5000);
         clearTimeout(); //Para um determinado intervalo, mas não zera o tempo.
         setTimeout(function() { 
            document.getElementById("time").innerHTML = "Tempo Parado!";
         }, 7000);
         
}

function stopCount() {
    clearTimeout(tempo);
    document.getElementById("time").innerHTML = "Parou a contagem!";
    setTimeout(function() {
        document.getElementById("time").innerHTML = "Tempo Parado!";
    }, 2000);
}

O que são Classe em JavaScript

Em 2015 foi introduzido ao JavaScript as Classes. As Classe são um conceito antigo em programação em várias linguagens, mas no Javascript é relativamente novo a ponto de muitos programadores terem dificuldades com elas.

Basicamente, Classe são como "fábricas" para criar objetos.

Pode-se dizer que são "funções especiais" para criar objetos.

Exemplo de Objeto
const carro = {marca: "Fiat", modelo: "Uno", ano:"2001"}

Assim como no mundo real as fábricas precisam de máquinas para produzirem, as Classe no javascript usam um método chamado constructor() para produzir objetos.

O nome da classe sempre deve ser escrito com a primeira letra maiúscula, invés de "class carro", se escreve "class Carro".

Valores alterados, aparecem posteriormente, a leitura no Javascript é de cima para baixo.
O mesmo é válido para a construção das classes, elas sempre devem ser feita antes das chamados, diferente de uma função.

No exemplo a seguir, foram criados três objetos com os valores já predefinidos.
A class Carros tem valores como marca, modelo e ano e 03 objetos chamados de veículo1, veículo2 e veículo3.

JS 

class Carro {
    constructor(vlr1, vlr2, vlr3) {
        this.marca = vlr1;
        this.modelo = vlr2;
        this.ano = vlr3;
        
    }
    buzina() {
        return this.modelo + " buzinou";
    }
}

const veiculo1 = new Carro("Fiat", "Palio", "2008")
const veiculo2 = new Carro("VW", "Gol", "2013")
const veiculo3 = new Carro("GM", "Vectra", "1994")

console.log(veiculo1.modelo);
console.log(veiculo2.buzina());
console.log(veiculo3);

veiculo3.ano = 1996
console.log(veiculo3);

document.getElementById("classes").innerHTML = 
`${veiculo1.marca}, ${veiculo2.modelo}, ${veiculo3.ano}`;

function verComparativo() {
    let diaComp = parseInt(document.getElementById('compDia').value);
    let mesComp = parseInt(document.getElementById('compMes').value) - 1; // Mês começa do zero
    let anoComp = parseInt(document.getElementById('compAno').value);

Vamos entender de JSON

JSON significa JavaScript Object Notation, é basicamente uma forma de transformar objeto em texto e vicee versa.
Ele é utilizado para transmitir dados entre sistemas de forma simples, pois texto é o formato lido por praticamente toda linguagem de programação.

Para trabalhar com JSON no JavaScript usamos dois métodos:

JSON.parce() -> Converte texto no padrão JSON em objeto;
JSON.stringify() -> Converte objetos em texto no padrão JSON.

JSON é como um tradutor da qual transforma código em texto, e texto devolta em código.
Vamos criar um exemplo onde será convertido um objeto em texto.

JS 

const carro = {
    marca: "Fiat",
    modelo: "Uno",
    ano:"2001",
    motor:["1.4","1.0"]
}

let txtJSON = JSON.stringify(carro)

document.getElementById('jsontxt').innerHTML = (txtJSON);

function buscarCEP() {
    let input = document.getElementById('cep').value;

    const ajax = new XMLHttpRequest();
    ajax.open('GET', 'https://viacep.com.br/ws/'+ input +'/json/');
    ajax.send();

    ajax.onload = function(){
//    document.getElementById('txtajax').innerHTML = this.responseText;
    let objajax = JSON.parse(this.responseText);
    let logradouro = objajax.logradouro;
    let bairro = objajax.bairro;
    let cidade = objajax.localidade;
    let estado = objajax.estado;
    let regiao = objajax.regiao;
    let ddd = objajax.ddd;

    document.getElementById('txtajax').innerHTML = 
    "Endereço: " + logradouro + 
    "<br>Bairro: " + bairro + 
    "<br>Cidade: " + cidade + 
    "<br>Estado: " + estado + 
    "<br>Região: " + regiao + 
    "<br>DDD: " + ddd;
    }
}
